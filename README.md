# 🌸 Сверочный отчёт — Цветочный магазин

Streamlit-приложение для ежемесячной сверки данных Битрикс с таблицей продаж менеджеров.

## Как пользоваться

1. Откройте приложение по ссылке
2. Загрузите файл Битрикс (`Deal_*.xls`)
3. Загрузите таблицу продаж (`*.xlsx`)
4. Нажмите **«Сформировать отчёт»**
5. Скачайте готовый Excel

## Что делает отчёт

- Сравнивает суммы по каждой сделке
- Находит сделки которых нет в таблице продаж
- Находит записи в продажах которых нет в Битрикс
- Проверяет категории (цветы / клубника / доставка)
- Проверяет даты (с учётом частичных оплат)
- Формирует Excel с цветовой разметкой ошибок

## Структура выходного файла

- 🚨 **Ошибки** — все расхождения по всем менеджерам
- 📦 **Товар** — детализация по товарам из Битрикс
- Для каждого менеджера:
  - **Сравнение** — построчное сравнение
  - **Нет в Таблице** — сделки только в Битрикс
  - **Нет в Битрикс** — записи только в таблице

## Развёртывание

### Streamlit Community Cloud (бесплатно)

1. Создайте аккаунт на [github.com](https://github.com)
2. Создайте новый репозиторий и загрузите все файлы
3. Зайдите на [share.streamlit.io](https://share.streamlit.io)
4. Войдите через GitHub → Deploy an app
5. Выберите ваш репозиторий, файл `app.py`
6. Получите ссылку вида `https://yourapp.streamlit.app`

## Файлы проекта

```
├── app.py              # Streamlit интерфейс

```

## Безопасность

- Файлы обрабатываются в памяти, не сохраняются на сервере
- Каждый пользователь работает в изолированной сессии
- Данные никуда не передаются
